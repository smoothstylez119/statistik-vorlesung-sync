# LEARNING_SUMMARY_PROMPT

## Input

Du erhältst ausschließlich validierte kanonische Seiten und das Glossar/Formelverzeichnis.

## Aufgabe

Erstelle eine Lernzusammenfassung, die beim Lernen hilft, aber klar von der Originaltranskription getrennt ist.

## Regeln

- Keine neuen Inhalte aus externem Wissen ergänzen, außer explizit als allgemeine Erklärung markiert.
- Nutze nur bestätigte Formeln und Definitionen.
- Unsichere Inhalte nicht als sicher erklären.
- Erzeuge prüfungsnahe Zusammenfassungen: Begriffe, Formeln, typische Fragen, Mini-Beispiele.

## Ausgabe

```markdown
# Lernzusammenfassung Statistik

## 1. Kernthemen

## 2. Begriffe

## 3. Formeln

## 4. Tabellen/Diagramme verstehen

## 5. Typische Prüfungsfragen

## 6. Unsichere Stellen, die vor dem Lernen geprüft werden sollten
```
